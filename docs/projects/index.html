AskGiveOp [ Spigot | ^1.21 ]#

このプラグインはユーザーをOPにする時に複数のOPからの承認が必要にするプラグインです。

※新しくサーバーを作成する場合はプラグインを入れる前に事前にOPを付与しておいてください。

config.ymlから必要な最低承認数を設定できます。
